# 使用pymysql进行简单的增删改查

---
### 查

```
# -*- coding=utf-8 -*-

import pymysql

try:
    conn = pymysql.connect(host = 'localhost', port = 3306, db = 'Persons', user = 'root', password = 'password', charset = 'utf8')
    cursor = conn.cursor()
    cursor.execute('SELECT * FROM Persons;')
    result = cursor.fetchall()
    print(result)
    cursor.close()
    conn.close()
except Exception as ept:
    print(ept.message)    
```

执行结果：

```
((1, 'Adams', 'John', None, 'London'),
 (2, 'Bush', 'George', 'Fifth Avenue', 'New York'),
 (3, 'Carter', 'Thomas', None, 'Beijing'))
```

### 改

```
# -*- coding=utf-8 -*-

import pymysql

try:
    conn = pymysql.connect(host = 'localhost', port = 3306, db = 'Persons', user = 'root', password = 'password', charset = 'utf8')
    cursor = conn.cursor()
    cursor.execute('UPDATE Persons SET CITY  = 'Shanghai' WHERE ID_P = 1;')
    conn.commit()
    cursor.close()
    conn.close()
except Exception as ept:
    print(ept.message)
```

### 增

```
# -*- coding=utf-8 -*-

import pymysql

try:
    conn = pymysql.connect(host = 'localhost', port = 3306, db = 'Persons', user = 'root', password = 'password', charset = 'utf8')
    cursor = conn.cursor()
    cursor.execute("INSERT INTO Persons VALUES (DEFAULT, 'Bush', 'John', 'Changan Street', 'Beijing');")
    conn.commit()
    cursor.close()
    conn.close()
except Exception as ept:
    print(ept.message)    
```

### 删

```
# -*- coding=utf-8 -*-

import pymysql
```
